## ğŸ“– User Guide

### Tab-by-Tab Guide

#### 1. Overview Tab

-   **Purpose:** Introduction and key metrics
-   **Features:** 3 value boxes showing latest index, year, and country
    count
-   **Use Case:** Dashboard orientation

#### 2. GCC Overall Tab

-   **Purpose:** View GCC aggregate performance
-   **Features:**
    -   Gauge chart with current index and delta
    -   Dimension bar chart
    -   6 dimension score cards
    -   Overall trend line
-   **Use Case:** Quick snapshot of GCC integration

#### 3. GCC Timeseries Tab

-   **Purpose:** Analyze trends across dimensions
-   **Features:**
    -   Multi-dimension selector
    -   Interactive timeseries chart
    -   Radar chart for latest year
    -   Correlation matrix
-   **Use Case:** Understanding dimension relationships and trends

#### 4. Country Profiles Tab

-   **Purpose:** Deep dive into individual countries
-   **Features:**
    -   Country selector
    -   4 value boxes (index, rank, level, YoY change)
    -   Overall trend chart
    -   Dimension scores (current)
    -   All dimensions over time
    -   Country vs GCC comparison
-   **Use Case:** Detailed country analysis

#### 5. Country Heatmap Tab

-   **Purpose:** Visual comparison across countries and dimensions
-   **Features:**
    -   Multi-select countries and dimensions
    -   Year range slider
    -   Country-dimension heatmap
    -   Rankings by dimension
-   **Use Case:** Spotting patterns and leaders/laggards

#### 6. GCC Analytics Tab

-   **Purpose:** Analyze changes and contributions
-   **Features:**
    -   Year range selector
    -   YoY changes in overall index
    -   YoY changes by dimension
    -   Contribution analysis
    -   Annual change trends
    -   Dimension change heatmap
-   **Use Case:** Understanding drivers of change

#### 7. Data Explorer Tab

-   **Purpose:** Access and export underlying data
-   **Features:**
    -   Dataset selector
    -   Interactive filterable table
    -   Export to CSV/Excel/Clipboard
    -   Column sorting and searching
-   **Use Case:** Data extraction for external analysis
